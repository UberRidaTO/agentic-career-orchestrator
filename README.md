# agentic-career-orchestrator
An automated, local pipeline (MacOS Cron) that orchestrates web-scraping and semantic analysis via the Claude 3.5 API. It performs a multi-variable gap analysis—filtering for values, geographic impact, and fiscal requirements—to deliver daily executive summaries and AI-driven resume tailoring logic.
